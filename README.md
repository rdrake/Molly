# Molly #

Molly is a system for automated knowledge discovery.

## Usage ##

This project uses [Lein](https://github.com/technomancy/leiningen) as its build system.

### Install dependencies ###

    lein deps

### Build Index ###

    lein run -m --database <path-to-db> --index <idx-folder>

### Run Tests ###

    lein spec
