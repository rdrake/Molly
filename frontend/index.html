About

Project Molly is a showcase of the cumulation of my Masters of Science thesis research. If you have an interest in the code, it is publically accessible on Github.

If you have an interest in what others in my group are doing, check out the site of my supervisor, Dr. Ken Pu.

This site's frontend is built on Twitter's Bootstrap, jQuery, jQuery UI, and KnockoutJS. The backend is mainly Clojure with Compojure handling routing, Jetty serving the dynamic information, and nginx serving static files and acting as a proxy to Jetty.
